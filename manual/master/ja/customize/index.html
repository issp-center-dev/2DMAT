(開発者向け)ユーザー定義アルゴリズム・ソルバー¶

本ソフトウェアは、順問題のためのソルバー Solver と最適化のためのアルゴリズム Algorithm を組み合わせることで全体の逆問題を解きます。 Solver と Algorithm はすでに定義済みのものがいくつかありますが、これらを自分で定義することで、適用範囲を広げられます。 本章では、 Solver や Algorithm を定義する方法およびこれらを使用する方法を解説します。
